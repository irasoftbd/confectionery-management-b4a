# 🍰 Confectionery Management System - B4A (Basic4Android)

একটি সম্পূর্ণ **Android অ্যাপ্লিকেশন** যা **B4A (Basic4Android)** দিয়ে তৈরি, যা কনফেকশনারি (মিষ্টি দোকান) ব্যবসা পরিচালনার জন্য ডিজাইন করা হয়েছে।

## 📋 প্রধান বৈশিষ্ট্য (Features)

### 🏪 **Inventory Management (ইনভেন্টরি ব্যবস্থাপনা)**
- ✅ নতুন পণ্য যোগ করুন
- ✅ পণ্য তথ্য সম্পাদনা করুন
- ✅ পণ্য মুছে ফেলুন
- ✅ স্টক পর্যবেক্ষণ এবং আপডেট
- ✅ পণ্য বিভাগ ম্যানেজমেন্ট
- ✅ পণ্য অনুসন্ধান এবং ফিল্টার
- ✅ কম স্টক সতর্কতা

### 💰 **Sales Management (বিক্রয় ব্যবস্থাপনা)**
- ✅ নতুন বিক্রয় অর্ডার তৈরি করুন
- ✅ গ্রাহক তথ্য সংরক্ষণ
- ✅ বিক্রয় বিবরণী তৈরি
- ✅ পেমেন্ট পদ্ধতি নির্বাচন (নগদ, চেক, ক্যার্ড)
- ✅ বিক্রয় ইতিহাস রক্ষা
- ✅ অর্ডার ট্র্যাকিং সিস্টেম
- ✅ দ্রুত পণ্য যোগ করার সুবিধা

### 📊 **Reports & Analytics (রিপোর্ট এবং বিশ্লেষণ)**
- ✅ দৈনিক বিক্রয় রিপোর্ট
- ✅ মাসিক বিক্রয় বিশ্লেষণ
- ✅ বার্ষিক বিক্রয় সংক্ষিপ্ত
- ✅ স্টক রিপোর্ট
- ✅ সবচেয়ে বেশি বিক্রিত পণ্য
- ✅ লাভজনকতা বিশ্লেষণ
- ✅ গ্রাহক তথ্য রিপোর্ট

### 💾 **Database Management (ডেটাবেস ব্যবস্থাপনা)**
- ✅ SQLite Local Database
- ✅ ডেটা ব্যাকআপ এবং রিস্টোর
- ✅ স্বয়ংক্রিয় ডেটা সংরক্ষণ
- ✅ ডেটা ভ্যালিডেশন

### ⚙️ **অন্যান্য ফিচার**
- ✅ ব্যবহারকারী-বান্ধব ইন্টারফেস
- ✅ বাংলা ভাষা সমর্থন
- ✅ অফলাইন মোডে কাজ করার ক্ষমতা
- ✅ ডার্ক মোড সাপোর্ট

## 📦 প্রাক-প্রয়োজনীয়তা (Prerequisites)

আপনার কম্পিউটারে থাকতে হবে:

1. **B4A IDE** - [ডাউনলোড করুন](https://www.b4x.com/b4a.html)

3. **Android SDK** - B4A সেটআপের সাথে অন্তর্ভুক্ত
4. **B4A Community License** - ফ্রি রেজিস্ট্রেশন (ঐচ্ছিক)
5. **Git** - (ঐচ্ছিক, কিন্তু সুপারিশকৃত)

## 🚀 দ্রুত শুরু করুন

### ধাপ ১: Repository ক্লোন করুন

```bash
# HTTPS এর মাধ্যমে
git clone https://github.com/irasoftbd/confectionery-management-b4a.git
cd confectionery-management-b4a

# অথবা SSH এর মাধ্যমে
git clone git@github.com:irasoftbd/confectionery-management-b4a.git
cd confectionery-management-b4a
```

### ধাপ ২: B4A IDE খুলুন এবং প্রজেক্ট লোড করুন

1. **B4A IDE** চালু করুন
2. **File** → **Open** ক্লিক করুন
3. `Code/Main.b4a` ফাইল নির্বাচন করুন
4. সম্পূর্ণ প্রজেক্ট লোড হবে

### ধাপ ৩: কম্পাইল করুন

```
Ctrl + B (Windows/Linux) বা Cmd + B (Mac)
```

অথবা **File** → **Build** → **Debug APK** নির্বাচন করুন।

### ধাপ ৪: অ্যাপ চালান

- এক্সটার্নাল ডিভাইস কানেক্ট করুন অথবা এমুলেটর চালু করুন
- B4A এ Run বাটন চাপুন বা `Ctrl + R`

## 🎯 ব্যবহারের ধাপ (Step-by-Step Usage)

### প্রথম চালু করার সময়:

1. **অ্যাপ খুলুন** → Dashboard দেখবেন
2. **Inventory সেটআপ করুন** → পণ্য বিভাগ তৈরি করুন
3. **পণ্য যোগ করুন** → নাম, মূল্য, স্টক লিখুন
4. **বিক্রয় শুরু করুন** → নতুন অর্ডার তৈরি করুন
5. **রিপোর্ট দেখুন** → বিক্রয় বিশ্লেষণ করুন

### প্রতিদিনের কাজ:

- সকালে: ইনভেন্টরি চেক করুন
- পুরো দিন: বিক্রয় রেকর্ড করুন
- সন্ধ্যায়: রিপোর্ট দেখুন এবং ডেটা ব্যাকআপ নিন
