## ✅ পূর্বশর্ত

আপনার কম্পিউটারে থাকতে হবে:

- **Windows 7+** অথবা **Mac OS 10.12+** অথবা **Linux**
- **Java Development Kit (JDK) 8+** - [ডাউনলোড](https://www.oracle.com/java/technologies/javase/jdk11-archive-downloads.html)
- **2GB RAM** (সর্বনিম্ন) অথবা **4GB+ RAM** (সুপারিশকৃত)
- **100MB ফ্রি ডিস্ক স্পেস**
- **ইন্টারনেট সংযোগ** (প্রথম সেটআপের জন্য)

## 🔧 B4A ইনস্টলেশন

### ধাপ ১: B4A ডাউনলোড করুন

1. [B4A অফিশিয়াল ওয়েবসাইট](https://www.b4x.com/b4a.html) ভিজিট করুন
2. **Download B4A** বাটন ক্লিক করুন
3. আপনার অপারেটিং সিস্টেম অনুযায়ী ভার্সন নির্বাচন করুন:
   - Windows: `B4A_v<version>_Installer.exe`
   - Mac: `B4A_v<version>_MacOS.zip`
   - Linux: `B4A_v<version>_Linux.zip`

### ধাপ ২: B4A ইনস্টল করুন

**Windows এ:**
1. ডাউনলোড করা `.exe` ফাইল ডাবল ক্লিক করুন
2. ইনস্টলেশন উইজার্ড অনুসরণ করুন
3. **Install Location**: `C:\B4A\` (ডিফল্ট ঠিক আছে)
4. ইনস্টলেশন সম্পন্ন হওয়ার জন্য অপেক্ষা করুন
5. **Finish** বাটন ক্লিক করুন

**Mac এ:**
1. `.zip` ফাইল এক্সট্র্যাক্ট করুন
2. ফোল্ডার এক্সটেনশনের জন্য চেক করুন
3. প্রয়োজনে Finder এ ড্র্যাগ করুন

**Linux এ:**
```bash
unzip B4A_v<version>_Linux.zip
cd B4A
./b4a
```

### ধাপ ৩: B4A প্রথম চালু করুন

1. B4A আইকন ডাবল ক্লিক করুন
2. প্রথমবার চালুর সময় কিছু সেটআপ হতে পারে
3. B4A IDE উইন্ডো খুলবে

### ধাপ ৪: Java এবং Android SDK সংক্রান় করুন

B4A প্রথম চালুতে:

1. **Tools** → **Configure Paths** খুলুন
2. **JDK Path** সেট করুন (যেখানে Java ইনস্টল আছে)
   - Windows: সাধারণত `C:\Program Files\Java\jdk1.x.x`
   - Mac: `/Library/Java/JavaVirtualMachines/jdk1.x.x/Contents/Home`
3. **Android SDK**: B4A স্বয়ংক্রিয়ভাবে সংক্রান় করবে
4. **OK** ক্লিক করুন

## 📂 প্রজেক্ট সেটআপ

### ধাপ ১: Repository ক্লোন করুন

```bash
# HTTPS এর মাধ্যমে
git clone https://github.com/irasoftbd/confectionery-management-b4a.git

# অথবা ম্যানুয়াল ডাউনলোড
# GitHub এ যান → Code → Download ZIP
```

### ধাপ ২: পরিকল্পনা তৈরি করুন

B4A তে প্রজেক্ট পরিকল্পনা তৈরি করা হয়:

```bash
# ক্লোন করা ফোল্ডারে যান
cd confectionery-management-b4a

# B4A প্রজেক্ট ফোল্ডার তৈরি করুন (যদি না থাকে)
mkdir -p Confectionery_Project
```

### ধাপ ৩: B4A তে প্রজেক্ট খুলুন

1. **B4A IDE** খুলুন
2. **File** → **Open** ক্লিক করুন
3. `Code` ফোল্ডারে নেভিগেট করুন
4. **Main.b4a** ফাইল নির্বাচন করুন
5. **Open** ক্লিক করুন

প্রজেক্ট লোড হয়ে যাবে এবং আপনি সকল ফাইল দেখতে পাবেন।

### ধাপ ৪: ডেটাবেস সেটআপ করুন

প্রথমবার চালুর সময় ডেটাবেস স্বয়ংক্রিয়ভাবে তৈরি হয়।

যদি ম্যানুয়াল সেটআপ করতে চান:

1. `Database/schema.sql` ফাইল খুলুন
2. SQL কমান্ড অনুসরণ করুন (B4A ডেটাবেস তৈরিতে সাহায্য করবে)

## 🎮 প্রথম চালু

### ধাপ ১: কম্পাইল করুন

B4A IDE তে:

```
Ctrl + B (Windows/Linux)
Cmd + B (Mac)
```

অথবা **File** → **Build** → **Debug** নির্বাচন করুন।

কম্পাইলিং প্রক্রিয়া শুরু হবে এবং কমান্ড লাইনে বার্তা দেখবেন।

### ধাপ ২: এমুলেটর বা ডিভাইস সংযোগ করুন

**Android এমুলেটর ব্যবহার করুন:**

B4A এর সাথে Android Virtual Device আছে:

1. **Run** (বা Ctrl+R) ক্লিক করুন
2. এমুলেটর চালু হবে (প্রথমবার ২-৩ মিনিট লাগতে পারে)
3. অ্যাপ্লিকেশন এমুলেটরে ইনস্টল হবে

**বা ফিজিক্যাল ডিভাইস:**

1. Android ফোন USB দিয়ে কম্পিউটারে সংযোগ করুন
2. **Developer Mode** চালু করুন:
   - Settings → About Phone → Build Number (৭ বার ট্যাপ করুন)
   - Settings → Developer Options → USB Debugging চালু করুন
3. **Run** ক্লিক করুন
4. অ্যাপ ইনস্টল এবং চালু হবে

### ধাপ ३: অ্যাপ্লিকেশন ব্যবহার করুন

অ্যাপ চালু হওয়ার পর:

1. **স্বাগত স্ক্রিন** দেখবেন
2. **Dashboard** এ প্রধান মেনু থাকবে
3. **Inventory** ট্যাব থেকে পণ্য যোগ করুন
4. **Sales** ট্যাব থেকে বিক্রয় রেকর্ড করুন
5. **Reports** ট্যাব থেকে বিশ্লেষণ দেখুন

## 🛠️ সাধারণ সমস্যা এবং সমাধান

### সমস্যা ১: B4A খোলা যাচ্ছে না

**সমাধান:**
```
1. JDK সঠিকভাবে ইনস্টল হয়েছে তা নিশ্চিত করুন
2. Environment Variable PATH তে Java যোগ করুন
   
Windows:
- Settings → System → Advanced System Settings
- Environment Variables → New
- Variable name: JAVA_HOME
- Variable value: C:\Program Files\Java\jdk1.x.x
```

### সমস্যা ২: কম্পাইলেশন ত্রুটি

**সমাধান:**
```
1. সকল ফাইল সঠিক ফোল্ডারে আছে তা চেক করুন
2. B4A IDE টুলস → Configure Paths পুনরায় সেট করুন
3. প্রজেক্ট ক্লিন করুন: File → Clean Project
4. পুনরায় কম্পাইল করুন
```

### সমস্যা ৩: ডেটাবেস লোড হচ্ছে না

**সমাধান:**
```
1. ডিভাইস স্টোরেজ পর্যাপ্ত আছে তা নিশ্চিত করুন
2. অ্যাপ অনুমতি চেক করুন: Settings → Apps → ফাইল সংরক্ষণ অনুমতি
3. ডেটাবেস ফাইল মুছে নতুন তৈরি করুন
```

### সমস্যা ৪: এমুলেটর শুরু হচ্ছে না

**সমাধান:**
```
1. ভার্চুয়ালাইজেশন চালু আছে তা নিশ্চিত করুন
2. BIOS সেটিংস চেক করুন (Intel VT-x বা AMD-V)
3. এমুলেটর মেমরি বাড়ান: Tools → Emulator Settings
4. B4A রি-স্টার্ট করুন
```

### সমস্যা ৫: অ্যাপ ক্র্যাশ হচ্ছে

**সমাধান:**
```
1. B4A এ Logcat দেখুন ত্রুটি বুঝতে
2. মোবাইল থেকে অ্যাপ ডেটা মুছে ফেলুন
3. অ্যাপ আনইনস্টল করে নতুন ইনস্টল করুন
4. GitHub Issues এ রিপোর্ট করুন
```

## 🔧 উন্নত কনফিগারেশন

### APK তৈরি করুন (Release Build)

আপনি যখন অ্যাপ অন্যদের দিতে চাইবেন:

1. **File** → **Build** → **Release APK** ক্লিক করুন
2. কিস্টোর ফাইল তৈরি করুন (প্রথমবার)
3. APK ফাইল তৈরি হবে
4. APK ডাউনলোড করুন এবং ইনস্টল করুন

### অ্যাপ স্বাক্ষর করুন

```
1. File → Create Release Build
2. Enter password and alias information
3. Wait for signing process
4. APK ready to distribute
```

### ব্যাকআপ এবং রিস্টোর

**ডেটা ব্যাকআপ:**
```
1. অ্যাপ খুলুন
2. Settings → Backup ক্লিক করুন
3. ব্যাকআপ ফাইল সংরক্ষিত হবে
```

**ডেটা রিস্টোর:**
```
1. Settings → Restore ক্লিক করুন
2. ব্যাকআপ ফাইল নির্বাচন করুন
3. ডেটা রিস্টোর হবে
```
